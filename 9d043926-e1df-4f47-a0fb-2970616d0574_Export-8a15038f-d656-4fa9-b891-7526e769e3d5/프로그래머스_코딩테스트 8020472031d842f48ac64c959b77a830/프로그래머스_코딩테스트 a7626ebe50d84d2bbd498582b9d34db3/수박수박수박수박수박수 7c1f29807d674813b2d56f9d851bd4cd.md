## 내가 푼 코드

```python
def solution(n):
    return '수박'* (n//2) if n % 2 == 0 else '수박'* (n//2) + '수'
```

## 타인이 푼 코드

### 타인이 푼 방법 1 - 파이썬 문자열 곱하기와 인덱싱을 활용

```python
def solution(n):
    str = "수박"*n
    return str[:n]
```

코드는 단순하나 효율성은 잘 모르겠음

### 타인이 푼 방법 2 - 간단한 코드

```python
def solution(n):
    return "수박" * (n//2) + "수" * (n%2)
```

나랑 코드는 동일하나 수를 붙이는 것에 더 간단하게 구현하여 if문을 사용안해도 되게 만듬
