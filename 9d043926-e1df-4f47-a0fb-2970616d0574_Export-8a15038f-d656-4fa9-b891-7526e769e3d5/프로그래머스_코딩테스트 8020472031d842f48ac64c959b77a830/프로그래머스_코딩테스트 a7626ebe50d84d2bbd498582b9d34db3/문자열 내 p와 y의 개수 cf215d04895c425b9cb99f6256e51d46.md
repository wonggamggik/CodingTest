# 문자열 내 p와 y의 개수

난이도: Level 1
링크: https://school.programmers.co.kr/learn/courses/30/lessons/12916?language=python3
문제 푼 날짜: 2023/07/11
해결 여부: 문제 해결

## 내가 푼 코드

```python
def solution(s):
    return True if s.lower().count('p') == s.lower().count('y') else False
```

## 타인이 푼 코드

### 타인이 푼 방법 1 - 간단한 코드
