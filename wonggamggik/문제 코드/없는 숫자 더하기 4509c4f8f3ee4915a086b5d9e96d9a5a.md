# 없는 숫자 더하기

난이도: Level 1
문제 링크: https://school.programmers.co.kr/learn/courses/30/lessons/86051?language=python3
문제 푼 날짜: 2023/07/19
해결 여부: 문제 해결

## 내가 푼 코드

```python
def solution(numbers):
    return 45 - sum(numbers)
```

## 타인이 푼 코드

### 타인이 푼 방법 1 - lambda를 이용한 풀이

```python
solution = lambda x: sum(range(10)) - sum(x)
```
