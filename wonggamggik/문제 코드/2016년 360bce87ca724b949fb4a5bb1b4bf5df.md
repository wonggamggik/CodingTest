## 내가 푼 코드

```python
def solution(a, b):
    month = [0,31,29,31,30,31,30,31,31,30,31,30,31]
    week = ["THU","FRI","SAT","SUN","MON","TUE","WED"]
    
    day = sum(month[:a]) + b
    
    return week[day%7]
```
