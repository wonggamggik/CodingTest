## 내가 푼 코드

```python
def solution(n):
    answer = 1
    
    for i in range(3,n+1,2):
        flag = True
        for j in range(2,int(i**(0.5))+1):
            if i % j == 0:
                flag = False
                break
        if flag == False:
            continue
        else:
            answer += 1
        
    return answer
```

## 타인이 푼 코드

### 타인이 푼 방법 1 - 에라토스테너스의 체를 구현

```python
def solution(n):
    num=set(range(2,n+1))

    for i in range(2,n+1):
        if i in num:
            num-=set(range(2*i,n+1,i))
    return len(num)
```

1. 숫자들을 집합으로 구성
2. i의 2배를 시작으로 끝까지 i의 배수들을 모두 지운다.
3. 이를 반복
