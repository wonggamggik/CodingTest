## 내가 푼 코드

```python
def solution(a, b):
    return sum([a[i]*b[i] for i in range(len(a))])
```

## 타인이 푼 코드

### 타인이 푼 방법 1 - zip() 함수를 이용한 풀이

```python
def solution(a, b):
    return sum([x*y for x, y in zip(a,b)])
```
