## 내가 푼 코드

```python
def solution(arr, divisor):
    answer = []
    for i in arr:
        if i % divisor == 0:
            answer.append(i)
    
    answer.sort()
    
    return answer if len(answer) != 0 else [-1]
```

## 타인이 푼 코드

### 타인이 푼 방법 1 - 코드 단순화

```python
def solution(arr, divisor): return sorted([n for n in arr if n%divisor == 0]) or [-1]
```
